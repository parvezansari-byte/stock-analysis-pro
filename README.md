# Stock Analysis Pro App V1

A professional Streamlit app for:
- Fundamental analysis
- Technical analysis
- Stock scoring model
- Buy / Hold / Avoid decision framework

## Features
- Fundamental Score (/100)
- Technical Score (/100)
- Combined Final Score
- Support / Resistance
- RSI / MACD / EMA / SMA
- Copy-ready client summary

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
